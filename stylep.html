Database

A database is an organized collection of data , It is the collection of schemas, tables and other object.

A database management system (DBMS) is a computer software application that interacts with the user
other applications, and the database itself to capture and analyze data.

Well-known DBMS include MySQL, Ingres, Microsoft SQL Server , Oracle, SAP HANA, and DB2.
